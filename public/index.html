GET Alle producten ophalen
GET /api/products

Query Parameters:

limit (optioneel) - Aantal resultaten (standaard: 10)
offset (optioneel) - Startpositie (standaard: 0) 
{
  "data": [
    {
      "id": 1,
      "name": "Gouden ring",
      "description": "18k gouden ring met diamant",
      "price": 1299.99,
      "stock": 5,
      "category_id": 1
    }
  ],
  "pagination": {
    "limit": 10,
    "offset": 0,
    "total": 25,
    "hasMore": true
  }
}
GET Producten zoeken
GET /api/products/search?q=ring

Query Parameters:

q (verplicht) - Zoekterm voor naam of beschrijving
GET Specifiek product ophalen
GET /api/products/:id
POST Nieuw product toevoegen
POST /api/products

Request Body:

name (verplicht) - Naam van het product
description (optioneel) - Beschrijving
price (verplicht) - Prijs (positief getal)
stock (optioneel) - Voorraad (standaard: 0)
category_id (optioneel) - Categorie ID 
{
  "name": "Zilveren ketting",
  "description": "925 sterling zilver",
  "price": 89.99,
  "stock": 15,
  "category_id": 2
}
PUT Product updaten
PUT /api/products/:id

Request Body: Alle velden zijn optioneel

DELETE Product verwijderen
DELETE /api/products/:id
